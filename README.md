## Project Description
This project program to find the maximum element in a matrix

## How to Compile and Run
1. Open a terminal.
2. Navigate to the folder containing the `MaxElementlnMatrix.java` file.
3. Compile the file using the command:  
   `javac MaxElementlnMatrix.java`
4. Run the program using the command:  
   `java MaxElementlnMatrix`
